Elvis Blog API

Welcome to the Elvis Blog API. This API is a simple RESTful API that allows you to create, read, update and delete blog posts. You can also get all blog posts or a single blog post by its ID.

The API is built using Node.js, Express.js and MongoDB. The API is hosted on Heroku and the database is hosted on MongoDB Atlas.

Visit the Blog's UI to interact with the API and get the full experience.
